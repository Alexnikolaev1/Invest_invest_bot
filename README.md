# INVESTMIND AI

Персональный AI-финансовый аналитик и наставник по инвестициям в Telegram.

## Возможности
- `/learn` — обучающий курс с уроками (Gemini) и квизами
- `/analyse <тикер>` — экспресс-анализ акции (yfinance/Alpha Vantage) или крипты (CoinGecko) + график за год
- `/news <тикер/тема>` — новости с AI-анализом тональности
- `/portfolio` — тест на риск-профиль и модельный портфель
- `/trade` и `/journal` — дневник сделок, P&L, диаграммы
- `/settings`, `/progress` — личный кабинет
- Голосовой ввод (Groq Whisper) и озвучка ответов (Edge TTS)

## Переменные окружения

Обязательные:
- `TELEGRAM_BOT_TOKEN` — токен бота от @BotFather
- `GEMINI_API_KEY` — ключ Google AI Studio (бесплатный тир Gemini)
- `GROQ_API_KEY` — ключ Groq (для распознавания голоса)

Рекомендуется для личного бота:
- `ALLOWED_USER_ID` — ваш Telegram user id; если задан, бот отвечает только указанным
  пользователям. Один id: `123456789`, несколько через запятую: `123456789,987654321`
  (узнать id: [@userinfobot](https://t.me/userinfobot))

Опциональные (фолбэки и расширения):
- `ALPHA_VANTAGE_API_KEY` — фолбэк для данных по акциям (25 запросов/день)
- `GNEWS_API_KEY` — новости (100 запросов/день)
- `NEWSAPI_KEY` — альтернативный источник новостей
- `WEBHOOK_URL` — если задан, бот работает через вебхук; иначе — поллинг
- `PORT` — порт для вебхука (Railway задаёт автоматически)
- `GEMINI_MODEL` — модель Gemini (по умолчанию `gemini-2.0-flash`)
- `DB_PATH` — путь к файлу SQLite (по умолчанию `investmind.db`)

## Локальный запуск (поллинг)

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
export TELEGRAM_BOT_TOKEN=...
export GEMINI_API_KEY=...
export GROQ_API_KEY=...
python bot.py
```

## Деплой на Railway

1. Создайте новый проект на Railway, подключите репозиторий с этим кодом.
2. В Variables добавьте переменные окружения из списка выше.
3. Railway автоматически определит `Procfile` (`web: python bot.py`).
4. Задайте `WEBHOOK_URL` равным публичному домену вашего сервиса
   (например, `https://<project>.up.railway.app`) — тогда бот перейдёт
   в режим вебхука. Если переменная не задана, бот будет работать
   через поллинг (подходит для тестирования, но не рекомендуется
   для продакшена на Railway).

## Важно

Бот — образовательный и аналитический инструмент. Он не даёт
инвестиционных рекомендаций и не несёт ответственности за финансовые
решения пользователей. Соответствующий дисклеймер выводится при
`/start` и в каждом ответе на `/analyse`.
